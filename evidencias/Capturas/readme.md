# Capturas de Evidencia para preparación de la Exposición

## Descripción
Esta carpeta reúne las capturas de pantalla obtenidas durante el desarrollo de la exposición sobre herramientas de **Model-Driven Development (MDD)**. Las imágenes sirven como evidencia visual del funcionamiento de las herramientas presentadas y complementan la documentación del proyecto.

## Contenido
En esta carpeta se incluyen:
- **16 capturas de pantalla (.png):** muestran diferentes momentos de las demostraciones realizadas durante la exposición, incluyendo la instalación y configuración de StarUML, la creación del modelo de clases del dominio Gimnasio, la generación automática de código en C#, la verificación del código generado y el proceso de roundtrip engineering (código antes y después de modificar el modelo).
- **README.md:** documento que describe el propósito y contenido de esta carpeta.

## Objetivo
Las capturas permiten evidenciar el proceso práctico realizado con StarUML, facilitando la comprensión de las funcionalidades presentadas y respaldando la información expuesta en las diapositivas y el informe.

## Nota
Las imágenes fueron organizadas como evidencia del trabajo realizado y tienen únicamente fines académicos y de documentación del proyecto.
