## Exposición

Esta carpeta contiene el material utilizado durante la exposición del proyecto **Generación de código fuente a partir de modelos UML mediante StarUML**.

### Archivos incluidos

- **Presentación.pptx:** Presentación editable utilizada durante la exposición.
- **Presentación.pdf:** Versión en PDF de las diapositivas para su visualización.

### Contenido de la presentación

- Fundamentos de MDD (Model-Driven Development).
- ¿Qué es StarUML?
- ¿Por qué elegimos StarUML?
- Subsistema modelado del Sistema de Gestión para un Gimnasio.
- Modelo UML desarrollado.
- Ventajas y limitaciones de StarUML.
- Del modelo UML al código C#.
- Comparativa de funcionalidades.
- Demostración práctica.
- Conclusiones.
